CSS Preprocessor

Preprocessors like Less and Sass make CSS much nicer to work with.

  • Variables
  • Shared styles
  • Easy nesting (dangerous)

Normalize not reset

A reset CSS file is like nuking and rebuilding, you have to add in all basic styling that you would normally get from the browser. A normalize CSS file provides just enough CSS to get all browsers working on a baseline and results in a much less cluttered view of your styles in an inspector.
